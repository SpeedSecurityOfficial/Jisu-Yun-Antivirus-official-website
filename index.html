系统要求

极速云杀毒支持广泛的Windows系统，占用资源少，运行流畅

Windows 7+

2C2G 推荐配置

  • 操作系统： Windows 7/8/10/11 (32位/64位)
  • 处理器： 双核处理器 1.5GHz 或更高
  • 内存： 2GB RAM (推荐4GB或更高)
  • 硬盘空间： 500MB 可用空间
  • 网络： 需要互联网连接以获取更新和云扫描
  • 分辨率： 1024×768 或更高
